# #1 序章
*「这个地方」指 Garden*

我希望这片地方是只有几个或几十个人的小地方，可以和谐共处的花园。至于为什么是花园，那是因为我在有机会的时候会独自跑到带有花朵的寂静小角落沉思（第三人称视角来看应该是个弱智）。

我很希望能和别人敞开心扉的聊，但是我本身的性格就很诡异加上有一次被人莫名的怼一通后，我就有点不敢了。

再加上我想分享的东西往往都很怪，所以我会在那带有花朵的寂静小角落或者没有人能听到的地方自言自语，对我来讲很有意思。（毕竟我自己真的愿意倾听我的每一句话，没有别人会认真听的）有的时候我也会发仅自己可见的朋友圈。这些都是满足自己的分享欲吧。

我把这些奇奇怪怪的东西都写在这里了。如果你真的看完了这个弱智写完的东西的话，那我也很佩服。我敢写在这里是因为这是一个无人问津的小花园，没有人会来看。

这样我也可以在没人看得到的情况下说点什么啦（自言自语），也算是满足自己奇怪的分享欲吧。如果哪天来看我这个文档的人变多了，那我就要把这份文档改成私人可见了，毕竟我不想让别人听到我自言自语。

作为花园，我的每一个项目都由一种花命名，例如Proejct Iris和Project Daisy。如果我能坚持的话，我希望我能亲手打造一个我自己的小花园。

如果你真的读了，那我再次佩服你。你愿意倾听弱智，无论是否真的有在意一点。

# #4 牵制
不知道各位下没下过国际象棋。下棋的时候总能发现棋子和棋子之间是有牵连的。（我下的很烂。）

某个子的行动范围正好在我方另一个子上，这时候对方吃掉后者那个子（叫他被保护的棋子吧），会将对方自己陷入困境，可以说这种局面是一种保护。

如果这时保护他的棋子挪走了，自己也会很麻烦，所以他也是一种牵制，让自己陷入麻烦。

当然了，要是只有一个子的话，除非对面也只有少得可怜的子或者我方实力强大，否则自己的胜算一般是不大的。

如果把上述我方的子想成自己和自己的朋友，把对方想成生活中的各种杂七杂八的事情，道理是类似的。

人一方面能享受到朋友所带来的利益（利益这个词情感比较怪），但是另一方面也要承担它所带来的麻烦。

除非你生活什么事都没有，或者你实力超强，否则你不太可能不交朋友，因为生活的压力没有朋友很难办。

如果是生活什么事都没有，那未免是生活太空虚了罢；如果是实力超强，看上去会是怪胎，只不过人们会因为其实力而在一定限度内宽容。

所以交朋友对我来讲是一个我很搐头的问题。生活里那些活泼的人可以是十分烦人的，那些安静的人可以是十分高冷的，虽然不是说不要太挑或者人都很讨厌，但是你总要接受这种潜在的可能性。

写到这里其实语言有点乱了，我想表达的并不是朋友很讨厌什么的，我想说生活里两面性总是持续的影响着心态和选择。

有关于牵制的东西可以说很多很多。例如生物。

人永远是人，也就意味着，人永远是生物，人也永远会收到生物本能的牵制。

人总有食欲、睡欲和性欲。这些欲望其实各个都是有点讨厌的。吃少了会饿，吃多了会胖；睡少了会困，睡多了又丧失了宝贵的时间；性欲更给我一种烈火燃烧但又有些丑陋的感觉。

没办法呀，人嘛。七情六欲，欲望总要满足，情感也总是越过理性，让我们做出不明智之选。

甚至是疾病与死亡，他们都是不可避免的。得了渐冻症，再多的思绪也只能转转眼球；失去视力，再美的日暮也不得见到；丢了性命，就和世界长辞了。

听上去很讨厌的，为什么我要有这般讨厌的躯体呢。

但你想，生命是一切奇迹的基础。我们能与朋友交流，我们能书写文字，我们能看朝阳升起，我们能为方程求解。没有生命的话其实什么都干不了。

说的奇怪点的话，能活就挺不错。

我们在短暂的生命中总会经历许多的磨难，但我们发挥出我们最耀眼的光彩，这不正是生命的赞歌吗？

所以好好珍惜生命，在不多的日子里享受每一天吧。

# #5 偏见和矛盾
说到LGBT，这个群体本身是没什么问题的，但是作为少数群体，逐渐沦为政客的工具，我觉得是很可悲的。无论是对于这个组织还是里面的人。

里面的人是无辜的，他们本身是没什么问题的。但是与社会不和导致的偏见，实际上导致了他们心里一定程度上是有点扭曲的。

心里的扭曲，导致他们的言行也是比较奇怪的。（这里的意思不是他们是怪胎，而是说他们的言行较常人更难以接受。）

厌恶某种程度上是这么来的。但是他们的行为又是自己所决定的。另一方面这样的组织举行大规模的游行和欢庆，也和这个组织的成员与外界的矛盾有关系。

我开始还是尊重他们的。但是后来的接触，我感觉他们有点恐怖。我不能说他们能代表集体，但是他们的行为很大程度上会影响我对集体的认识。

他们受到的排挤使得他们更加恐惧别人，恐惧别人又会让他们在找到愿意接纳他们的人时更加信任。能使他们信任的人是少见的（因为他们是受偏见的），故他们可能会将自己压抑的情感更多的释放。

我很难说那是性欲还是爱欲，但是无论哪个他们都是具有一定侵犯性的。不断的在信任的人身上索要更多，这是一种伤害，不是么。

我是这么理解他们的行为模式的，所以我觉得他们是可怜而可悲的。我并不支持去消灭这样的组织或者人，只希望什么时候能平等吧。

# #6 苦与乐
我曾经遇到过一位老师。他性格十分开朗，也读过很多书，人很潇洒。

我不展开介绍他，但是他跟我说过的一句话令我印象深刻。

> 乌托邦不存在。

还是那样，人总归是生物，所以一定遵循生物的本能。当一样欲望被满足，人会本能地寻求更多。这是生物本能之一吧。

因此，当你有低期望或者欲望时，一件小事便可以让你开心。但当一直收到刺激的时候，你的期望阈值就会上升，原先的小事已经不能满足你了，能刺激你的东西就需要更刺激。

所以一直这样，人只会一直想要更多。但是现实中不可能一直刺激你，这时候一些小事情就可能苦了。

苦和乐总是相对的。你一直苦，某件小事就能乐；你一直乐，某件小事就能苦。乌托邦这样的极乐环境是不存在的，不可能会有一样东西一直满足你。

生活不能一味的追求乐（当然不要自讨苦吃）。乐总要付出什么，不然他总要发展成贪婪的欲望。好比说鸦片，极小的付出和强烈的刺激让人迅速成瘾，又因刺激强筋而使人快速想要更多。这样一来，生活中的平凡事都变成苦了，唯有手头的冰是乐。

这样的生活不乏味吗？毒品又真的是乐吗？我看到有人深入探究过量服药所带来的刺激，并不断说明他与毒品的区别。但那和毒品，在本质上有何不同呢。

其实换个角度想，苦亦是乐，乐亦是苦。生活总要这样下去，所以不如乐观的活着吧。

# #7 生肉
前几篇都挺沉重，今天想写点带点抽象色彩的事情。

写的可能有点恶心。

冬季时，空气湿度会大幅降低，使得空气干燥无比。

皮肤在这种干燥的环境下，加上我不怎么喝水，就开裂了。

嘴唇那块的肉就裂开了，流着少量的血液，结成了痂，像一道印记般不偏不倚的在下嘴唇的中点处。

这个倒不是重点。好玩的是：

当这个地方因为某些原因破的时候，里面的血会自然的流出，而又会被舌头上的味蕾所感知到。

那种滋味，新鲜的血液散发着一种血红蛋白的独特气味，甚至有一点点铁的味道（大抵是错觉）。

血液的气息，又伴随着组织的味道，交织在我的口腔内，仿佛在品味生肉。

青嫩的生肉组织，配合上流淌的血液所散发的难以描述的那种气味，刺激着嗅觉（味觉？嗅觉？分不清。）。

如果加上一些幻想色彩的话，那时应该像是唤起什么体内先前行为习惯一样啊呜啊呜要吃生肉？

不过智人（至少我）没有这种感觉，我只感觉到我仿佛像学会用火以前的人，唤起什么隐隐约约的感觉，但对于吃生肉一点欲望没有。

很神奇的感觉。分享出来又很奇怪，兴许还很恶心。我觉得倒还好，但是读者就不一定了。

所以写这里了。这件事情怎么说也挺抽象的。

# #9 树苗

不过这不是重点。其实除了前三篇自成一体外，后面这些篇都互相有联系。没有哪个主题是太独立的。

孩子呱呱坠地的那天开始，就要开始认识世界了。那时的孩子，不具有任何能力，也不具有任何责任。

生命基本物质的进出都需要监护人去管理，更别提什么责任了。

就这样，无忧无虑的活着（除了生病什么的），再大一点，可能开始学走路，监护人耐心的、一步一步的辅导着，给予充足的鼓励……

能力越来越多，也要逐渐开始放手。开始去上学，去学算术、去背古诗、去交朋友，什么什么的。

幼儿和小孩期间，认识到的东西其实很少、很局限。知道基本饱和，觉得自己差不多什么都懂了。

那就差不多是青春期了。

青春期的生理发展倒是没什么意思。我认为心理才算是真正的青春期。

能想的东西愈发变多，也不禁会想到，“人活着干什么？”“我从哪里来？从何而去？”

这种问题曾困扰我数个月，再结合青春期前期压力会强于此前人生中任何一部分，不免会迷茫。

我认为，直到这些问题真正的被抛出，真正的经历了探寻问题的过程（可能会迷茫），最终得到了自己对于这些问题的理解，这才是真正青春期。

写这篇文章的时候因为日程安排被打断了，其实核心观点就是那样。

# #10 空虚
总感觉空虚感一直“游离”在我周围。

空虚感时强时弱，和朋友玩兴许空虚感会弱很多，敲代码的时候专注在工作上也会弱很多。但是这都说不定呢？

开心的时候，说不定突然想到，一切都会走向消失。没准也会突然感到空虚呢？

不排除不应期这种生理上的因素。这种空虚的来源很重要的一部分来自于心理。

蔚蓝档案里面是有提到过关于虚无主义的：

> 一切皆为徒劳。无论去向何处，终将化为虚无。

原文是拉丁文，我也不知道他具体的来源。但是这其实就是虚无主义的表现。不管你怎样去行动，人类总有一天会消失，你干的所有事情都没有什么用，反正都是消失。

小的时候，青春期刚刚开始的时候，突然想到，“我活着来干什么呢？”

这个问题对我十分重大。我从未想过这种问题，感到深深的无力。迷茫扑过来。此前一直在走的道路，如今仿佛是摘下眼罩，发现前面根本没有路标。

就这样，我开始漫无目的的思考。在探索中，最终得到的答案是：“探索”。

我主张人应当是自由的（对于自由的定义之后有机会再说）。人被抛下来，没有被赋予任何意义。

从细胞体开始对外界作出反应开始，生命就已经诞生了。化合物聚在一起本来就没什么意义，和柳絮在水面上漂无大区别。但是生命就是这样诞生了。（我印象里是这样。）

所以人到底去要干什么是没有什么官方答案的。答案纯粹是要自己去找的。

既然地球来都来了，苦难也不可避免，为什么不去探索这个世界奇特的地方，去了解世界？所以我得出了“探索”。

后来啊，上了初中，交了新朋友，学了高级的数学，看到了更广阔的东西。这种空虚感消退了很多，我对死亡的恐惧，可以算是对生活的热爱的最直接体现了。

现在又对死没什么感觉了。

在觉得日子没意思的时候，干什么都没太多动力。哪怕是下楼走一圈，或是动一下鼠标，甚至是重新上床，感觉都很累。感觉日子毫无意思。活着很无聊的那种感觉。

曾经的迷茫，算是对人本身的探索。

现在的迷茫，更多要聚焦在自己身上了。“我要干什么？”“我要去哪里？”“我有什么价值？”

6个月后就要去南半球了。我连未来想去干什么都不知道。

你不觉得很可悲吗？从前都是“啊，下个学期还是在学校上学呢”这样下去，一直下去，从来没有什么未来的人生规划。

有些人有理想无经济能力，而我拥有这些能力却又没有什么规划。

我不知道路在何方。我不知道船舵该往哪边摆。甚至在社会的风向下，帆船摇摇晃晃。

真正强化这种情感的可能是寒假可悲的计划。我甚至要在为数不多的假期内继续按照时刻表执行，表面上的”商讨“实际并无余地。我甚至要看今天是星期几，是不是法定节假日。

我是员工吗？我不是啊！但是在就这样点时间内还有去执行这种计划，我实在是讨厌这样牵着脖子走。这完全背离我的哲理。

我害怕自己丧失自己的价值，我害怕自己被人讨厌，被人摒弃。我追求独立，但我却又总是想写点什么出来。

我有什么能力？
